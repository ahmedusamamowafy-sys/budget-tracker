⬆️ Import Expenses
Paste tab-separated rows below (copied from Excel or the Export tab). Each row must have:

Main Category Sub Category Date Amount Description Payment

Date format: YYYY-MM-DD — columns 2 and 3 can be the same Sub Category.
ℹ️ Column Order
Col 1 — Main Category
Col 2 — Category (can = Sub Category)
Col 3 — Sub Category ← used for matching
Col 4 — Date (YYYY-MM-DD)
Col 5 — Amount (number)
Col 6 — Description (optional)
Col 7 — Payment Method
⬇️ Export
📋 Copy for Excel
Select a month → Copy → Paste into your Excel Expenses sheet. Includes a "Type" column (Spendable / Saving Boxes).
🗑️ Clear Month Data
Remove all expenses for a month after pasting into Excel.
ℹ️ Monthly Workflow
1. Log expenses daily here
2. Export → Copy → Paste into Excel
3. Click Close Month macro in Excel
4. Clear this month → fresh start ✅
